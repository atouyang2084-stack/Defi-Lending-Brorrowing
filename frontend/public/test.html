3. 常见问题排查

如果前端没有反应：

  1. 按 F12 打开开发者工具
  2. 查看 控制台(Console) 标签页
  3. 查看 网络(Network) 标签页
  4. 检查是否有红色错误信息
  5. 刷新页面查看错误

检查步骤：

1. 检查Hardhat节点是否运行: npx hardhat node
2. 检查合约是否部署: npx hardhat run contracts/script/deploy.js --network localhost
3. 检查测试账户: npx hardhat run scripts/setup-test-account-fixed.js --network localhost
4. 检查前端服务器: cd frontend && npm run dev
5. 访问: http://localhost:5174

4. Repay问题特别说明

重要: InsufficientLiquidity() 错误 不可能 来自repay函数！

这个错误只会在 borrow 函数中发生。

如果repay时看到这个错误，说明：

  • 前端调用了错误的函数（调用了borrow而不是repay）
  • 或者用户实际上在尝试借款而不是还款
